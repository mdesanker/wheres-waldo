# Where's Waldo

A Where's Waldo browser game created with React

Live demo [here](https://mdesanker.github.io/wheres-waldo)

## How to setup

```
git clone https://github.com/mdesanker/wheres-waldo.git
cd wheres-waldo
npm install
npm start
```

## Built with

- React
- React Router
- styled-components
- Firebase - Firestore
